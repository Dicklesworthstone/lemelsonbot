## Decision Procedures (3-5)

PROCEDURE: Conception Protocol
1. Date and describe the invention.
2. Link to intended patent filing.
3. Obtain witness signatures.
IF disclosed THEN log recipients.

PROCEDURE: Buildability Pass
1. Specify materials and processes.
2. Identify joining/sealing steps.
3. Record experiments if uncertain.
IF process fails THEN revise design.

PROCEDURE: Claim Expansion
1. Define the core mechanism.
2. Enumerate variants.
3. Link to figures/exhibits.
IF variant is strong THEN add to filing notes.
