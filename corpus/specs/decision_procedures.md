PROCEDURE: Conception-to-Evidence
1. Write a dated Record of Conception entry.
2. State mechanism, intended use, and intent to patent.
3. Add manufacturing feasibility notes.
4. Obtain witness signatures with dates.
IF disclosure occurred THEN add disclosure log entry.

PROCEDURE: Partner Outreach Loop
1. Identify target partner or manufacturer.
2. Prepare a model or concise summary.
3. Visit or present and record interest level.
4. Request samples, quotes, or agreements.
5. Log follow-up date and next step.
IF partner interest is weak THEN pivot to alternate partner.

PROCEDURE: Sample Validation Pass
1. Acquire samples or materials.
2. Define the test or evaluation criteria.
3. Run experiments and log results.
4. Update design or process notes.
IF sample fails THEN seek alternative materials or vendors.

PROCEDURE: Claim Expansion Pass
1. Define the core mechanism.
2. Enumerate variants (materials, shapes, contexts).
3. Assign figure references for each variant.
4. Cross-link to prior filings/exhibits.
IF a variant is strong THEN include in application notes.

PROCEDURE: Continuity Trace
1. Link the current entry to relevant application numbers/exhibits.
2. Note changes or improvements from prior filings.
3. Record attorney interactions or filing status.
IF amendments are pending THEN log timeline and next steps.
