# Lemelson Protocol: Evidence Pack v0.1

> **Purpose**: Store external evidence beyond the quote bank

## Evidence Record Format

```
EV-001
Title: [Paper/Patent/Note]
Source: [URL or file path]
Accessed: [YYYY-MM-DD]
Summary: [1-2 sentences]
Excerpts:
  E1: "..." (page/section)
  E2: "..." (page/section)
Tags: [manufacturing, testing, disclosure]
```

## Citation Rules

- Cite full record as `EV-001`
- Cite excerpt as `EV-001#E1`
- If no evidence record exists, mark as `inference`
